Social Sparks

SocialSparks is an Android App created for my friend Cora that encourages her to stay in touch with loved ones throughout the day. The app is written in Kotlin and for each time of day it suggests one small action to connect with someone.


What it does

Time Click uses a unique interface that lists times of the day as clickable radio buttons. Click a time, and then click the button to get a social suggestion, based off of what's going on at that time. If you forget to select a time, it will give you an error message. You can also start over from the beginning with the reset button.


Times and suggestions

Morning - Send a Good Morning message to a family member.
Mid-Morning — Send email to a friend to say thank you!
Be Afternoon - Share a funny meme or interesting link with a friend.
Afternoon Snack Time — Send a Thinking of you message.
Call a friend or relative to say hello for 5 minutes.
After Dinner / Night — Write a sticky note to someone you appreciate.


How I built it

This program was written using Kotlin in the Android Studio program. A radio group was used for picking the time because that only allows you to choose one time at a time. The logic for suggesting was based off of when statements and the suggested response is based off of the time you selected. I put in input validation so that even though the program does not do anything with your input, it will behave properly even if you did not select a time. I ensured that in the event that the user just submits without selecting the various options provided- that an error message appears. I also put in a lot of comments throughout the program so that you can see what the program is doing and for mental notes for myself to keep track of whether a section of the program has been completed or not.



How to run it

1. Clone the repo
2. Open in Android Studio
3. Run on an emulator or your phone
